This is .NET Core Web Api project. This project fetches top stories from Hacker News API. This project implements Api: GetAll - stories/items returned from Hacker News API. This project is using built-in feature of .NET Core for Dependency Injection & Error Handling. This project is using Swashbuckle.AspNetCore nuget for swagger.

GetAll Api: This api returns list of NewsItems stored in the cache.
